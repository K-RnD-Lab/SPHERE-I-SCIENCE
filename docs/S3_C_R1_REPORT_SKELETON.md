# S3-C-R1 Report Skeleton

## Title

Crop Stress And Support Profiles

## Research home

- Sphere: `S3`
- Lane: `S3-C Crop Stress & Yield Systems`
- Research line: `S3-C-R1`

## Research question

How can crop stress signals be organized into interpretable profiles that connect drought, heat, salinity, and related pressures with support factors that matter for field resilience and yield protection?

## Scope

This first report should stay narrow:

- one crop-stress evidence table
- one stress signal panel
- one short support-factor shortlist

## Data included

- crop context
- stress type
- signal category
- growth-stage context
- yield-linked effect
- support factor
- interpretation notes

## Methods

### Source collection

- select recent reviews and high-value comparative studies
- extract structured crop-stress records

### Standardization

- normalize stress types
- normalize signal classes
- normalize support factors
- normalize yield-facing outcomes

### Ranking logic

Support-factor ranking should combine:

- recurrence across sources
- field relevance
- crop-context plausibility
- yield-protection logic

## Outputs

- normalized evidence table
- stress signal panel
- support-factor shortlist
- one concise report
- one or two figures

## Limits

- crop heterogeneity
- limited direct field validation for some support factors
- uneven comparability across signal classes
- no universal recommendation claims

## Conclusion

The first report should end with:

- which stress signals look most consistently useful
- which support factors appear strongest
- what needs crop-specific validation next
