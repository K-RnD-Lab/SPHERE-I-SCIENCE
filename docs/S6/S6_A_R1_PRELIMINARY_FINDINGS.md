## Scope v0

This first execution pass narrows `S6-A-R1` to:

- regional sorting-readiness logic
- recovery vs disposal interpretation
- facility-registry context
- modeled material recovery and avoided CO2e assumptions

## Current patterns

- the implemented app already makes `S6-A-R1` the flagship S6 environmental platform layer
- official waste metrics are useful for regional comparison
- facility registry data adds infrastructure context but not full operational capacity
- avoided CO2e should stay visibly modeled and assumption-based

## Practical interpretation

The strongest immediate report is not a broad climate claim.

The strongest report is:

- which regions appear stronger or weaker by sorting-readiness indicators
- where recovery and landfill-disposal signals diverge
- which assumptions drive the modeled climate-benefit layer
- what data gaps matter before policy-level claims

## Current discipline

At this stage we may say:

- SortSmart can support public-interest environmental interpretation
- official open data can produce useful regional signals
- modeled recovery potential is a scenario layer

At this stage we should not say:

- that the app is an official environmental monitoring system
- that avoided CO2e is directly measured
- that facility registry presence proves real operating capacity
