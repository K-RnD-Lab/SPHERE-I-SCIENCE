# S4-A-R2 Report Skeleton

## Working title

State-Specific Metabolomic Profiles: Comparing Healthy, Perturbed, And Multi-Group Biochemical States

## Research home

- Sphere: `S4`
- Lane: `S4-A Metabolomics & Signature Discovery`
- Research line: `S4-A-R2 State-Specific Metabolomic Profiles`

## Question

How can metabolomic data be organized into defensible state-specific profiles across healthy, perturbed, and subgroup comparisons?

## Scope

- healthy baseline variation
- perturbed state profiles
- subgroup and multi-condition comparison
- shared vs distinct metabolite-state logic
- confounder and platform caution

## Data included

- healthy adult profile studies
- disease or perturbation profile studies
- multi-group comparison studies
- platform-comparison caution studies

## Methods

- extract group comparisons
- normalize profile logic across studies
- distinguish shared vs distinct signals
- annotate likely confounders
- summarize profile interpretability

## Outputs

- state-profile evidence table
- shared-vs-distinct profile panel
- confounder note layer
- compact interpretation summary

## Figures

- healthy vs perturbed profile comparison
- shared vs distinct metabolite-state logic
- confounder and platform caution matrix

## Limits

- cohort-specific variation
- heterogeneous platforms
- inconsistent annotation depth
- non-transferability across matrices or populations

## Conclusion

State-specific metabolomics is useful when it is treated as profile interpretation with context, not as flat biomarker extraction.
