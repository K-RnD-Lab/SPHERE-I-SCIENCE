# S4-E-R1 Source Registry

## Purpose

This file records the first evidence layer for:

- `S4-E-R1 Candidate Chemical Prioritization`

It is meant to support:

- translational candidate ranking
- pathway-to-candidate reasoning
- mechanism-aware shortlist logic
- caution around weakly supported chemical inference

## Active scope

Current scope is deliberately narrow:

- candidate chemical prioritization from metabolic evidence
- translational logic built on pathways and targets
- critical filtering of weak network-pharmacology-style overreach

## Current research stance

At this stage, `S4-E-R1` is not trying to validate compounds.

It is trying to distinguish:

- candidates with stronger metabolomic, pathway, and target support
- candidates that are plausible but weakly grounded
- shortlist logic worth carrying into later deeper studies

## What to collect next

Next source additions should prioritize:

- papers with explicit candidate shortlists rather than only pathway claims
- studies that tie metabolites, targets, and compounds together clearly
- critical methodological papers that expose overclaiming
- examples where candidate ranking changed after stronger evidence integration

## Claim discipline

At this stage we may say:

- candidate-prioritization is a valid translational layer
- stronger shortlists need pathway and target support, not only compound presence
- methodology critique is part of the value of `S4-E`

At this stage we should not say:

- that shortlist ranking equals therapeutic validity
- that network pharmacology by itself is enough
- that candidate chemicals are ready for intervention use
