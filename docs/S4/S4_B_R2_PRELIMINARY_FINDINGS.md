## Current patterns

- mechanism ranking needs more than pathway p-values
- method sensitivity is a real risk in untargeted metabolomics
- activity inference and cross-omics support can strengthen pathway claims
- large cohort resources help identify recurring metabolic patterns, but not direct causality

## Practical interpretation

The useful output is a ranked mechanism-priority table with visible uncertainty.

The strongest records should combine:

1. coherent metabolite pattern
2. defensible annotation
3. method-stable pathway support
4. cross-omics or external biological support

## Current discipline

At this stage we may say:

- `S4-B-R2` can prioritize candidate mechanisms
- weak pathway enrichment should be filtered before interpretation
- method notes are part of the scientific output

At this stage we should not say:

- that a pathway list proves mechanism
- that a single enrichment tool is enough
- that mechanism priority equals validation
