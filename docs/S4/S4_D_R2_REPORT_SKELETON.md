# S4-D-R2 Report Skeleton

## Working title

Shared Mechanism Interpretation Across Biological Contexts

## Research home

- Sphere: `S4`
- Lane: `S4-D Cross-Domain Mechanism Models`
- Research line: `S4-D-R2 Shared Mechanism Interpretation`

## Question

Which recurring biochemical motifs are transferable enough to guide cross-domain hypotheses?

## Scope

Include:

- motif-level comparison
- domain context
- shared vs specific signal
- transferability score

Exclude:

- direct domain substitution
- exact biomarker universalism
- application claims without context

## Methods

1. collect motif evidence across domains
2. classify motif type
3. compare shared and context-bound parts
4. assign transferability score
5. write boundary notes

## Outputs

- shared motif table
- transferability ranking
- context-boundary report
- candidate cross-domain hypothesis list

## Figures

- motif transfer matrix
- domain-context comparison table
- shared-vs-specific signal map

## Limits

- recurrence is not identity
- organism and tissue context matter
- shared motifs still need domain-specific validation

## Conclusion

`S4-D-R2` should make cross-domain mechanism reasoning useful and safe by keeping context boundaries explicit.
