# S4-E-R2 Report Skeleton

## Working title

Evidence-Based Chemical Plausibility For Translational Shortlists

## Research home

- Sphere: `S4`
- Lane: `S4-E Translational Chemical Inference`
- Research line: `S4-E-R2 Evidence-Based Chemical Plausibility`

## Question

Which candidate chemicals are plausible enough to remain in a translational shortlist?

## Scope

Include:

- candidate evidence layers
- pathway support
- target support
- method-risk flags
- validation need

Exclude:

- efficacy claims
- unranked candidate catalogs
- network-pharmacology-only conclusions

## Methods

1. collect candidate records
2. classify evidence layers
3. score plausibility
4. flag weak method logic
5. define next validation move

## Outputs

- plausibility evidence table
- comparative shortlist
- weak-claim filter
- validation-priority report

## Figures

- candidate plausibility matrix
- evidence-layer stack
- shortlist comparison chart

## Limits

- plausibility is not efficacy
- target prediction is not mechanism validation
- natural-product evidence often has variable source quality

## Conclusion

`S4-E-R2` should make translational chemical inference more defensible by filtering candidate shortlists through explicit evidence tiers.
