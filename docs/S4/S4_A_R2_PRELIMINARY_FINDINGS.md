# S4-A-R2 Preliminary Findings

## Research home

- Sphere: `S4`
- Lane: `S4-A Metabolomics & Signature Discovery`

## Scope v0

This first active execution pass narrows `S4-A-R2` to:

- healthy vs perturbed biochemical state profiles
- shared vs distinct profile interpretation across groups
- caution around cohort structure, platform differences, and confounders

That scope is intentionally tighter than a broad metabolomics review.

## Current state-profile shortlist

### Tier 1 patterns

- healthy metabolomic baselines already vary by age, sex, and matrix, so disease claims need baseline discipline
- multi-group metabolomics is most useful when it distinguishes shared and distinct profiles rather than only ranking one condition
- state profiles become stronger when they are interpreted as patterns, not just as isolated metabolites

### Tier 2 patterns

- platform differences can distort biomarker discovery and profile comparability
- unbalanced populations weaken downstream predictive claims
- some “state” profiles may reflect lifestyle, processing, or cohort composition more than biology of interest

## Why these patterns matter

They jointly show three things:

- `S4-A` can support profile-level biological interpretation, not only marker ranking
- state comparison is one of the cleanest bridges from signatures into pathways
- disciplined baseline and confounder handling is part of the method, not an optional note

## Practical interpretation

The current evidence already supports three real outputs:

1. a normalized state-profile comparison table
2. a shared-vs-distinct metabolite pattern panel
3. a short caution layer on confounders and platform sensitivity

## Current working logic

### 1. Healthy baselines are not metabolically flat

The healthy-adult profiling work makes it clear that age, sex, and sample matrix already shape metabolic profiles in nontrivial ways.

### 2. Multi-group studies are more informative than simple case-control thinking

The neurodegeneration and cardiometabolic comparison studies show that state-specific metabolomics becomes more useful when it separates common and distinct patterns across related conditions.

### 3. State profiles can be meaningful outside narrow disease framing

The healthy-lifestyle signature work supports the idea that profile logic is broader than pathology alone.

### 4. Technical caution must sit inside the method

Platform-comparison evidence makes it clear that weak comparability can distort downstream biomarker or profile claims.

## Current discipline

At this stage we may say:

- state-specific metabolomic profiles are useful when interpreted with baseline and subgroup discipline
- shared-vs-distinct profile logic is stronger than flat metabolite ranking
- confounder and platform awareness are part of profile validity

At this stage we should not say:

- that one profile automatically generalizes across populations
- that state separation alone proves mechanism
- that any classifier built on unbalanced or platform-mixed data is robust

## Immediate next step

The next execution move for `S4-A-R2` should be:

- expand the evidence table with a few additional strong multi-group and baseline studies
- keep only records that sharpen state-profile logic
- avoid bloating the pack with repetitive disease-biomarker overviews
