# S4-B-R1 Source Registry

## Purpose

This file records the first evidence layer for:

- `S4-B-R1 Pathway Perturbation Maps`

It is meant to support:

- pathway-ranking logic
- perturbation-map design
- confidence-aware pathway interpretation
- limits of metabolite-to-pathway inference

## Active scope

Current scope is deliberately narrow:

- pathway perturbation interpretation from metabolomic shifts
- methodological caution around pathway analysis
- examples where pathway structure is interpretable enough to use

## Current research stance

At this stage, `S4-B-R1` is not trying to produce a universal pathway engine.

It is trying to distinguish:

- strong perturbation patterns
- weak enrichment-only claims
- pathway maps worth carrying forward into later mechanism work

## What to collect next

Next source additions should prioritize:

- papers with explicit metabolite-to-pathway coherence
- studies that compare pathway shifts across conditions or tissues
- sources that show when enrichment signals fail
- mechanism papers that tie pathway perturbation to candidate targets or processes

## Claim discipline

At this stage we may say:

- pathway interpretation is useful only when coherence and annotation quality are considered
- pathway perturbation maps can be more informative than raw metabolite lists
- methodological caution is part of the scientific value of `S4-B`

At this stage we should not say:

- that every pathway-enrichment result is mechanistically valid
- that one applied paper proves pathway causality
- that metabolomics pathway maps fully resolve target-level biology
