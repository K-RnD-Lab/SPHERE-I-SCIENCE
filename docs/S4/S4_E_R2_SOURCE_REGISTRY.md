# S4-E-R2 Source Registry

## Purpose

This file records the first evidence layer for:

- `S4-E-R2 Evidence-Based Chemical Plausibility`

It supports:

- chemical plausibility scoring
- shortlist comparison
- filtering weak translational claims

## Current stance

`S4-E-R2` should be the filter between "interesting candidate" and "candidate worth deeper validation."

It should prefer:

- layered evidence
- explicit limits
- validation next steps

It should reject:

- unsupported network pharmacology
- mechanism overclaiming
- candidate lists without evidence ranking
