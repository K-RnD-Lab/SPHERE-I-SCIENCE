# S4-C-R1 Report Skeleton

## Working title

Target And Enzyme Prioritization From Metabolomic And Pathway Signals: A Practical Evidence Review

## Research home

- Sphere: `S4`
- Lane: `S4-C Enzyme, Target & Substrate Logic`
- Research line: `S4-C-R1 Target And Enzyme Prioritization`

## Research question

How can metabolomic and pathway signals be translated into defensible enzyme and target priorities that support later mechanism or translational work?

## Scope

This report covers:

- enzyme relevance ranking
- target shortlist logic
- pathway-to-target bridging
- ambiguity and confidence handling

This report does not attempt:

- full target validation
- therapeutic ranking across disease areas
- genome-scale exhaustive target discovery

## Data included

- computational enzyme-substrate inference papers
- metabolomics plus pathway plus target integration papers
- metabolomics studies linked to plausible target discovery

## Methods

1. collect evidence on target and enzyme inference from metabolite/pathway signals
2. normalize target-support fields
3. separate strong multi-signal support from weak speculative links
4. rank enzyme or target candidates by plausibility and interpretability

## Outputs

- one target evidence table
- one ranked enzyme shortlist
- one pathway-to-target bridge panel
- one concise report

## Figures

Possible figures:

1. pathway-to-target bridge map
2. enzyme relevance ranking panel
3. target confidence matrix

## Limits

- many target hypotheses remain computational or integrative rather than validated
- mapping between metabolites and enzymes is often one-to-many
- translational relevance may vary by domain context

## Conclusion

The report should conclude with:

- which enzyme or target candidates are most plausible
- where target logic is still ambiguous
- what can be carried forward into translational chemical inference
