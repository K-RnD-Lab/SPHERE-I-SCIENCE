# S3-C-R1 Source Registry

## Research home

- Sphere: `S3`
- Lane: `S3-C Crop Stress & Yield Systems`
- Research line: `S3-C-R1 Crop Stress And Support Profiles`

## Purpose

This file is the first evidence layer for `S3-C-R1`.

It tracks:

- which stress and resilience sources we actually use
- which sources support signal interpretation versus support-factor logic
- how yield relevance is being grounded

## Current active scope

For the first execution pass, `S3-C-R1` is deliberately narrowed to:

- drought, heat, and salinity stress
- yield-relevant crop responses
- support factors that repeat across field-relevant literature

This is a better first scope than trying to cover every possible crop stressor.

## Current research stance

At this stage, `S3-C-R1` should be treated as:

- a crop-stress interpretation and support-factor project

not as:

- a full agronomic prescription system
- a generic stress-detection technology survey

## What to collect next

The next useful sources should include:

- crop-specific field studies for wheat, maize, and legumes
- additional multi-stress comparisons
- stronger support-factor studies for irrigation, soil amendments, and genotype-stage interaction

## Claim discipline

From this registry, we may claim:

- that drought, heat, and salinity repeatedly depress yield-related outcomes
- that stress profiles are crop-specific and trait-specific
- that support factors such as root traits, silicon, and multi-trait monitoring deserve higher attention

We should not yet claim:

- universal support-factor rankings
- crop-independent resilience recipes
- that every stress indicator is equally useful in field deployment
