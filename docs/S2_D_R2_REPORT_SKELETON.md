# S2-D-R2 Report Skeleton

## Title

Host-Pathogen Interaction Interpretation Across Resistant And Susceptible Plant States

## Research home

- Sphere: `S2`
- Lane: `S2-D Plant-Pathogen & Defense Systems`
- Research line: `S2-D-R2`

## Research question

Which host-side patterns most clearly distinguish resistant and susceptible plant states across bacterial and fungal disease systems?

## Scope

This first report should stay narrow:

- resistant vs susceptible host comparisons
- 5-7 cross-system patterns
- bacterial and fungal contexts only

## Data included

- plant system
- pathogen context
- comparison pattern
- pattern class
- host interpretation
- mechanism note
- evidence level

## Methods

### Source collection

- recent primary comparative studies
- at least one bacterial and two fungal contexts

### Standardization

- normalize comparison-pattern names
- separate resistance patterns from susceptibility/failure patterns
- assign evidence level and priority score

### Prioritization logic

Pattern ranking should combine:

- recurrence across pathosystems
- interpretive clarity
- host-side usefulness
- balance of positive and negative comparison patterns

## Outputs

- comparative evidence table
- resistant-vs-susceptible interpretation panel
- one figure of recurring host patterns
- one summary note on what most often separates resistance from vulnerability

- pathosystems differ strongly

The first report should end with:

- which host patterns recur most strongly
