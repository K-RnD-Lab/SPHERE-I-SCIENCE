# S2-E-R2 Report Skeleton

## Title

Comparative Translational Profiles For Plant-Derived Candidates

## Research home

- Sphere: `S2`
- Lane: `S2-E Translational Plant Bioactivity`
- Research line: `S2-E-R2`

## Research question

Which plant-derived candidates remain the strongest when mechanism opportunity is compared against translational friction?

## Scope

This first report should stay narrow:

- 5-6 benchmark candidates
- no disease-specific ranking
- comparison by opportunity, friction, and source clarity

## Data included

- compound
- primary plant source
- compound class
- mechanism breadth
- source clarity
- evidence maturity
- translational friction
- main limitation

## Methods

### Source collection

- use candidate-specific and comparative reviews
- capture both positive signal and translational limitation

### Standardization

- normalize candidate names and source plants
- assign opportunity and friction logic using transparent categories

### Prioritization logic

Candidate comparison should combine:

- breadth of mechanistic rationale
- confidence of source identity
- maturity of repeated literature presence
- level of translational friction

## Outputs

- comparative evidence table
- opportunity-versus-friction panel
- shortlist ranking note
- one figure comparing stronger and weaker translational profiles

## Figures

### Figure 1

Opportunity vs friction scatter or matrix

### Figure 2

Candidate profile comparison by class

## Limits

- scoring remains interpretive
- different compounds are studied in different contexts
- preclinical recurrence does not guarantee translational value

## Conclusion

The first report should end with:

- which candidates are strong despite friction
- which candidates are promising but premature
- which next evidence would change the ranking most
