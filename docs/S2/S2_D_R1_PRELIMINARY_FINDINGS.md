## Scope v0

This first active execution pass narrows `S2-D-R1` to:

- bacterial and fungal resistance contexts
- core defense signatures that can be reused across plant systems
- hormone, barrier, transcriptional, and metabolite outputs

This is a better first scope than trying to describe all plant immunity.

## Current compact panel

### Hormone and early-response branch

- salicylic acid accumulation / biosynthesis
- jasmonic acid plus ethylene network logic
- reactive oxygen species and H2O2 burst
- callose deposition

### Downstream output branch

- PR-gene and PR-protein maintenance
- receptor kinase and MAPK-linked defense activation
- phenylpropanoid / flavonoid / phenolic defense shifts
- camalexin regulation

## Why these signatures matter

They jointly cover the main layers of a usable defense panel:

- signal induction
- early barrier response
- transcriptional maintenance
- metabolite output

Without this layered structure, `S2-D-R1` would collapse into either:

- only hormone talk
- or only a list of resistance papers

## Interpretation

### Salicylic acid

- strongest broad anchor for biotrophic and hemi-biotrophic defense logic
- especially useful because it is supported both as a marker and as a biosynthetic pathway

### Callose

- one of the cleanest physical-barrier markers
- useful because its loss is directly tied to resistance loss in a mechanistic study

### PR genes and PR proteins

- good bridge between signaling and actionable transcript-based readouts
- especially valuable in resistant-vs-susceptible comparisons

### Ethylene

- stronger here as a context-dependent maintenance signal than as a standalone universal marker
- especially useful in bacterial wilt resistance logic

### Camalexin and phenylpropanoid outputs

- these keep the panel biologically honest
- plant defense is not only signaling; it also needs chemical outputs

## Current discipline

At this stage we may say:

- these signatures are strong enough to enter a first compact defense panel
- they recur across multiple plant-pathogen contexts
- they give `S2-D-R1` a real molecular identity

At this stage we should not say:

- that this is a universal immunity map
- that these signatures predict field resistance by themselves
- that all pathogen classes can be treated as equivalent
