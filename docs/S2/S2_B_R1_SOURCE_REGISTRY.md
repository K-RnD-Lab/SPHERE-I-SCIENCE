# S2-B-R1 Source Registry

## Research home

- Sphere: `S2`
- Lane: `S2-B Phytochemicals & Natural Products`
- Research line: `S2-B-R1 Phytochemical Atlas And Candidate Prioritization`

## Purpose

This file is the first real evidence layer for `S2-B-R1`.

It is not only a concept note. It is the registry that should track:

- which sources we actually use
- what each source contributes
- how strong the evidence is
- what can and cannot be claimed from it

## Current active scope

For the first execution pass, `S2-B-R1` is deliberately narrowed to:

- edible and medicinal plant-derived compounds
- oxidative-stress and inflammation modulation
- evidence-backed atlas ranking

This is a better first scope than trying to cover every plant compound class at once.

## Current research stance

At this stage, `S2-B-R1` should be treated as:

- an atlas-building and evidence-ranking project

not as:

- a validated therapeutic pipeline

## What to collect next

The next useful sources should include:

- plant family-specific phytochemical reviews
- species-to-compound databases
- high-quality compound-class reviews
- evidence-backed bioactivity summaries
- plant family-specific reviews that add non-polyphenol diversity

## Claim discipline

From this registry, we may claim:

- that a compound is reported
- that it belongs to a class
- that it has evidence-linked bioactivity claims
- that it is worth prioritizing for further interpretation

We should not yet claim:

- validated therapeutic effect
- clinical utility
- direct mechanism proof unless clearly supported
- that the current shortlist is exhaustive
