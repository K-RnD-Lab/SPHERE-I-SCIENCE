# S2-B-R1 Report Skeleton

## Title

Phytochemical Atlas And Candidate Prioritization

## Research home

- Sphere: `S2`
- Lane: `S2-B Phytochemicals & Natural Products`
- Research line: `S2-B-R1`

## Research question

How can plant-derived compounds be organized into a transparent atlas and prioritization layer that supports real biological interpretation rather than only a compound list?

## Scope

This first report should stay narrow:

- one plant family or one bioactivity theme
- one evidence table
- one ranked shortlist

## Data included

- species metadata
- compound identifiers
- compound class
- tissue context
- bioactivity theme
- evidence notes

## Methods

### Source collection

- literature-backed source selection
- structured extraction into a normalized evidence table

### Standardization

- normalize plant species
- normalize compound names
- normalize compound classes

### Prioritization logic

Candidate ranking should combine:

- evidence density
- compound class relevance
- bioactivity plausibility
- mechanism note strength

## Outputs

- atlas table
- ranked shortlist
- compound-class summary
- one or two figures

- evidence heterogeneity
- literature bias
