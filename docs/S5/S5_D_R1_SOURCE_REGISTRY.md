# S5-D-R1 Source Registry

## Purpose

This file records the first evidence layer for:

- `S5-D-R1 Resilience Marker Profiles`

It supports:

- cognitive reserve and brain maintenance logic
- pathology-vs-cognition interpretation
- protective profile design
- resilience vs risk distinction

## Active scope

Current scope is deliberately narrow:

- resilience to aging and neurodegenerative burden
- cognitive reserve and brain maintenance
- multimodal biomarkers of preserved cognition
- support-factor ranking without causal overclaim

## Current research stance

At this stage, `S5-D-R1` is not trying to prescribe resilience interventions.

It is trying to build:

- a protective-profile evidence structure
- a reserve and support-factor taxonomy
- a cautious way to compare risk burden with preserved function

## What to collect next

Next source additions should prioritize:

- cohorts with pathology plus cognitive performance
- studies distinguishing resilience from resistance
- multimodal imaging and biofluid evidence
- intervention studies only when endpoints are clear

## Claim discipline

At this stage we may say:

- resilience and reserve are useful constructs for explaining preserved cognition
- protective profiles require multimodal evidence
- cognitive function can be partly preserved despite pathology burden

At this stage we should not say:

- that reserve prevents neurodegenerative disease
- that lifestyle proxies prove biological mechanism
- that resilience profiles are ready as clinical tools
