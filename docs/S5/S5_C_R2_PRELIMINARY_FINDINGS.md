## Research home

## Scope v0

This execution pass focuses on:

- progression-associated markers
- shared vs disease-specific patterns
- marker role separation across neurodegenerative diseases

## Current patterns

- plasma proteomics can reveal both shared and disease-specific biology across AD, PD, and FTD
- NfL is valuable as a neurodegeneration and progression marker but has limited disease specificity
- GFAP can reflect astroglial activation and severity in selected contexts
- computational progression models are useful for temporal interpretation but require validation

## Practical interpretation

The first useful output is a matrix separating:

- diagnostic support
- severity marker
- progression marker
- shared degeneration marker
- disease-specific pattern

## Current discipline

At this stage we may say:

- progression and disease distinction are related but different tasks
- shared markers need specificity labels
- cross-disease comparison can improve mechanism framing

At this stage we should not say:

- that one blood marker defines one disease
- that disease overlap means identical biology
- that progression models are ready without cohort validation
