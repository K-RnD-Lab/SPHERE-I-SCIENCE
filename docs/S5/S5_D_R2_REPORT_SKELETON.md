# S5-D-R2 Report Skeleton

## Working title

Recovery Pattern Interpretation: Reserve, Biomarkers, Network Signals, And Post-Event Brain Outcomes

## Research home

- Sphere: `S5`

- Research line: `S5-D-R2 Recovery Pattern Interpretation`

## Question

Which signals help interpret recovery after brain injury, stroke, or acquired brain dysfunction?

## Scope

- cognitive reserve in recovery
- chronic TBI biomarkers
- post-stroke cognitive recovery
- neuroimaging and network markers
- prognosis and rehabilitation caution

## Methods

- classify event type
- separate reserve from recovery process
- record signal type and endpoint
- flag prognostic readiness

## Outputs

- recovery evidence table
- reserve-vs-recovery matrix
- biomarker role note
- recovery interpretation report

## Figures

- recovery phase map
- reserve vs recovery grid
- marker readiness ladder

## Limits

- event heterogeneity
- timing differences
- outcome variation
- prognostic overclaim risk

## Conclusion

Brain recovery interpretation is strongest when pre-event reserve, post-event biomarkers, recovery phase, and functional outcome are recorded separately.
