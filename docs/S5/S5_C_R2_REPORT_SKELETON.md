# S5-C-R2 Report Skeleton

## Working title

Progression And Pattern Distinction In Neurodegeneration: Shared And Disease-Specific Marker Logic

## Research home

- Sphere: `S5`
- Lane: `S5-C Neurodegeneration Models`
- Research line: `S5-C-R2 Progression And Pattern Distinction`

## Question

Which markers and model types help distinguish progression, severity, and disease-specific patterns across neurodegenerative conditions?

## Scope

- AD, PD, FTD, ALS, MSA, and related disorders
- plasma proteomics and blood biomarkers
- NfL, GFAP, tau, protein aggregation markers
- computational progression models

## Methods

- classify disease context
- separate marker role
- record progression vs distinction logic
- flag specificity limits

## Outputs

- progression evidence table
- shared vs disease-specific marker matrix
- severity vs progression note
- cross-disease comparison report

## Figures

- disease-by-marker matrix
- marker role ladder
- shared vs disease-specific mechanism panel

## Limits

- overlap across disease mechanisms
- assay and cohort heterogeneity
- disease-stage confounding
- limited availability of some imaging markers

## Conclusion

Neurodegeneration progression models are useful when marker role, disease context, and specificity limits are explicit.
