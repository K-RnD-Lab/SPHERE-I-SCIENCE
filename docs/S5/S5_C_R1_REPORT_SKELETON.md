# S5-C-R1 Report Skeleton

## Working title

Neurodegeneration State Models: Biomarker, Imaging, And Disease-Stage Logic Across Brain Disorders

## Research home

- Sphere: `S5`
- Lane: `S5-C Neurodegeneration Models`
- Research line: `S5-C-R1 Neurodegeneration State Models`

## Question

How can neurodegenerative disease states be organized using biomarker, imaging, pathology, and stage evidence?

## Scope

- AD biomarker staging
- fluid and imaging biomarker interpretation
- Lewy body and alpha-synuclein marker logic
- protein aggregate imaging
- digital biomarker caution

## Data included

- biomarker staging reviews
- neuropathology and biomarker reviews
- seeding-assay reviews
- PET imaging reviews
- plasma biomarker and digital biomarker reviews

## Methods

- extract biomarker class
- classify marker role
- separate disease state from clinical endpoint
- record readiness and cut-point limits

## Outputs

- disease-state evidence table
- biomarker role matrix
- stage-readiness note
- neurodegeneration state report

## Figures

- biomarker role ladder
- AD vs non-AD state model map
- diagnostic vs prognostic vs monitoring role grid

## Limits

- biomarker assay differences
- cutoff and gray-zone uncertainty
- disease heterogeneity
- clinical implementation gap

## Conclusion

Neurodegeneration state models are useful when biomarkers are assigned explicit roles rather than treated as interchangeable evidence.
