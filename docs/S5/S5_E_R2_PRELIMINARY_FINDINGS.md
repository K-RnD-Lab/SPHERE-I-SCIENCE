## Research home

- Sphere: `S5`

## Scope v0

This execution pass focuses on:

- early-warning interpretation
- progression monitoring
- time-based stratification

## Current patterns

- future-risk protein panels are promising for long-horizon dementia prediction
- p-tau217 combined with NfL or GFAP is more useful than isolated markers
- NfL is valuable for progression monitoring but not disease-specific
- computational progression models are useful for research interpretation but need validation before practical deployment

## Practical interpretation

The first useful output is a marker-role matrix:

- early warning
- diagnostic support
- progression monitoring
- trial enrichment

## Current discipline

At this stage we may say:

- longitudinal markers can support structured brain-risk interpretation
- time horizon and endpoint define marker meaning
- predictive models need readiness labels

At this stage we should not say:

- that future-risk markers diagnose future disease
- that model performance is enough for clinical use
- that one biomarker role covers all use cases
