# S5-B-R2 Source Registry

## Purpose

This file records the evidence layer for:

- `S5-B-R2 Contextual Vulnerability Interpretation`

It supports:

- layered cognitive vulnerability profiles
- risk-plus-reserve interpretation
- modifiable risk and support-factor mapping

## Current source set

### 1. Lancet Commission dementia prevention framework

Source:

- https://www.thelancet.com/commissions/dementia-prevention-intervention-care

Why it matters:

- strongest public-health risk-factor framework for dementia prevention
- gives `S5-B-R2` a life-course and modifiable-risk backbone

## Current stance

`S5-B-R2` is a profile interpretation pack, not a public diagnostic tool. It should keep population risk, individual vulnerability, and modifiability clearly separated.
