# S5-B-R1 Source Registry

## Purpose

This file records the first evidence layer for:

- `S5-B-R1 Cognitive Aging Risk Profiles`

It supports:

- vulnerability profile logic
- cognitive aging risk grouping
- biomarker and risk-factor separation
- reserve and vascular-context interpretation

## Active scope

Current scope is deliberately narrow:

- cognitive aging vulnerability profiles
- risk and protective domains
- biomarker-adjacent interpretation
- profile usefulness before clinical claims

## Current research stance

At this stage, `S5-B-R1` is not trying to diagnose dementia or predict individual outcomes.

It is trying to build:

- interpretable vulnerability profiles
- disciplined risk-domain categories
- evidence-aware cognitive aging comparisons

## What to collect next

Next source additions should prioritize:

- cohort studies with cognitive domain outcomes
- risk-factor meta-analyses
- imaging or blood-marker papers with clear cognitive endpoints
- studies that distinguish risk, resilience, and reserve

## Claim discipline

At this stage we may say:

- cognitive aging vulnerability is multidomain
- vascular, biological aging, reserve, and lifestyle signals can support profile logic
- profile interpretation is stronger than one-factor explanation

At this stage we should not say:

- that one marker predicts cognitive decline for everyone
- that aging clocks are ready as standalone clinical tools
- that risk-factor association equals direct intervention proof
