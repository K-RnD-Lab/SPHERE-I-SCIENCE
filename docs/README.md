# Active Docs

This folder is for active science work only.

Use `docs/` for:

- current lane maps
- current execution status files
- active research guides
- evidence schemas
- source registries
- report skeletons
- active transition notes for live research work

Do not use `docs/` as a graveyard for older planning layers.

Historical or superseded material should move to:

- `archived_docs/`

## Folder structure

`docs/` is now grouped by sphere plus one shared core layer:

- `docs/core/`
  - cross-sphere active execution notes
  - rollout notes
  - active sync or transition files
- `docs/S1/`
  - active biomedical and oncology research staging
- `docs/S2/`
  - active plant science and phytochemistry research staging
- `docs/S3/`
  - active agricultural biology and biofertilizer research staging
- `docs/S4/`
  - active biochemistry and metabolomics research staging
- `docs/S5/`
  - active neuroscience and aging research staging
- `docs/S6/`
  - active ecology and environmental science research staging

## Current active logic

Right now, `docs/` should contain the files that still directly support:

- active structuring of `S1-S6`
- active research staging
- current execution planning
- actual starter packs for upcoming studies

If a file is still being used to define, stage, validate, or execute a current study, it belongs here.
