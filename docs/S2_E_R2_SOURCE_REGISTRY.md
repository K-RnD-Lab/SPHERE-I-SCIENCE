# S2-E-R2 Source Registry

## Research home

- Sphere: `S2`
- Lane: `S2-E Translational Plant Bioactivity`
- Research line: `S2-E-R2 Comparative Translational Profiles For Plant-Derived Candidates`

## Purpose

This registry tracks the sources used to compare plant-derived candidates by translational opportunity and translational friction.

The goal is not to prove therapeutic efficacy.

The goal is to show, in a disciplined way:

- which candidates have broad mechanism coverage
- which candidates have stronger source identity
- which candidates are repeatedly limited by bioavailability, stability, or standardization

## Current active scope

For the first execution pass, `S2-E-R2` is narrowed to:

- the current `S2-B-R1` candidate pool
- comparative translational logic
- opportunity versus friction scoring

Evidence role:

Use in project:

Evidence role:

Source:

- https://www.mdpi.com/1467-3045/48/2/172

Evidence role:

- candidate recurrence and evidence-maturity framing

## Current stance

At this stage, `S2-E-R2` should be treated as:

- a comparative plausibility layer

not as:

- a therapeutic leaderboard
- a formulation recommendation sheet

## What to collect next

The next useful sources should include:

- one recent formulation-focused review
- one compound-standardization review
- one human-exposure or safety-oriented source for a non-mainstream candidate

## Claim discipline

From this registry, we may claim:

- that some candidates are easier to justify than others
- that translational friction is uneven across compounds
- that plant-derived promise does not equal translational readiness

We should not yet claim:

- clinical superiority
- validated dosing logic
- direct product readiness
