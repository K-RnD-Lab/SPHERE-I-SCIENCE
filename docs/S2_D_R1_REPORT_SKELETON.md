# S2-D-R1 Report Skeleton

## Title

Core Plant Defense Signature Panel

## Research home

- Sphere: `S2`
- Lane: `S2-D Plant-Pathogen & Defense Systems`
- Research line: `S2-D-R1`

## Research question

Which molecular and metabolite signatures are strong enough to form a compact, reusable plant defense panel across bacterial and fungal resistance contexts?

## Scope

This first report should stay narrow:

- bacterial and fungal contexts only
- 5-7 core signatures
- hormone, barrier, transcriptional, and metabolite layers

## Data included

- plant system
- pathogen context
- signature name
- signature class
- defense layer
- mechanism note
- evidence strength

## Methods

### Source collection

- recent primary open-access studies
- at least one bacterial and one fungal context

### Standardization

- normalize signature names
- separate hormone, barrier, transcriptional, and metabolite layers
- assign evidence level and priority score

### Prioritization logic

Signature ranking should combine:

- recurrence across systems
- clarity of mechanism
- usefulness as an interpretable marker
- balance across defense layers

## Outputs

- defense-signature evidence table
- ranked compact panel
- one layered figure of the signature system
- one shortlist of transferable signals for future studies

## Limits

- cross-species comparability is incomplete
- evidence depth differs by signature
- resistance markers are context-dependent

## Conclusion

The first report should end with:

- which signatures are strongest now
- where the evidence is still thin
- which next studies would strengthen the panel
