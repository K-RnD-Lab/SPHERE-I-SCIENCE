# S3-A-R2 Report Skeleton

## Title

Soil Microbiome State Classification

## Research home

- Sphere: `S3`
- Lane: `S3-A Soil & Rhizosphere Microbiomes`
- Research line: `S3-A-R2`

## Research question

How can soil and rhizosphere microbiomes be classified into interpretable “healthier,” “stressed,” or “response-sensitive” states without pretending that one universal microbiome signature applies across all crops and management systems?

## Scope

This first report should stay narrow:

- one microbiome-state evidence table
- one support-vs-risk indicator panel
- one crop-context comparison matrix

## Data included

- crop context
- management context
- state label
- microbiome indicator
- indicator direction
- support or risk interpretation
- generalizability note

## Methods

### Source collection

- select recent predictive, comparative, and management-linked sources
- extract structured microbiome-state records

### Standardization

- normalize crop and management labels
- normalize state classes
- normalize indicator classes
- normalize generalizability notes

### Classification logic

State classification should combine:

- management sensitivity
- stress linkage
- support-vs-risk direction
- predictive usefulness
- generalizability constraints

## Outputs

- normalized evidence table
- support-vs-risk indicator panel
- crop-context comparison matrix
- one concise report
- one or two figures

## Limits

- strong context dependence
- limited cross-system generalizability
- uneven field validation
- no universal soil-health scoring claim

## Conclusion

The first report should end with:

- which indicators look most useful
- where generalizability breaks down
- what should be validated in crop-specific follow-up work
