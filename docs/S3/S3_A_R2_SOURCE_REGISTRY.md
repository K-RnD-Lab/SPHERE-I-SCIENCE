# S3-A-R2 Source Registry

## Research home

- Sphere: `S3`
- Lane: `S3-A Soil & Rhizosphere Microbiomes`
- Research line: `S3-A-R2 Soil Microbiome State Classification`

## Purpose

This file is the first evidence layer for `S3-A-R2`.

It tracks:

- which microbiome-state sources we actually use
- which sources support classification versus prediction
- where generalizability is strong and where it clearly breaks

## Current active scope

For the first execution pass, `S3-A-R2` is deliberately narrowed to:

- healthy vs stressed soil-state logic
- crop-context and management-context differences
- response-sensitive soil states for microbial intervention

This is a better first scope than trying to invent one universal soil-health taxonomy.

## Current research stance

At this stage, `S3-A-R2` should be treated as:

- a microbiome-state interpretation project

not as:

- a universal soil-health certification model
- a claim that one indicator panel fits every crop and field

## What to collect next

The next useful sources should include:

- crop-specific disease-context microbiome comparisons
- additional management-comparison studies
- direct comparisons between organic, conventional, and diversified systems
- more field-scale examples with both microbiome and crop-performance outcomes

## Claim discipline

From this registry, we may claim:

- that microbiome states are shaped by management and stress
- that some indicators are useful for support, risk, or response-sensitive classification
- that generalizability is limited and must be labeled explicitly

We should not yet claim:

- one universal healthy-soil signature
- deployment-ready classification for all crops
- that diversity alone is enough to classify soil state
