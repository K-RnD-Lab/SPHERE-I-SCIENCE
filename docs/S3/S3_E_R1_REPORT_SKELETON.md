# S3-E-R1 Report Skeleton

## Title

Field Translation Playbooks

## Research home

- Sphere: `S3`
- Lane: `S3-E Field Translation & Practical Deployment`
- Research line: `S3-E-R1`

## Research question

How can agricultural biology and intervention findings be translated into field-ready playbooks that practitioners can actually use, adapt, and monitor instead of leaving them as research-only outputs?

## Scope

This first report should stay narrow:

- one translation evidence table
- one compact playbook template
- one shortlist of strong adoption and advisory patterns

## Data included

- target user
- intervention context
- advisory format
- monitoring checkpoints
- barriers and enabling factors
- translation notes

## Methods

### Source collection

- select recent work on advisory, adoption, and participatory translation
- extract structured field-translation records

### Standardization

- normalize target users
- normalize advisory formats
- normalize barrier and enabling-factor categories
- normalize monitoring checkpoints

### Translation logic

Playbook ranking should combine:

- practicality
- trust and advisory support
- adaptation to local context
- monitoring clarity
- likely adoption

## Outputs

- field-translation evidence table
- compact playbook template
- adoption and advisory shortlist
- one concise report
- one or two figures

## Limits

- strong dependence on local institutions and trust structures
- uneven transferability across geographies
- limited crop-specific validation at this stage
- no claim of universal playbook success

## Conclusion

The first report should end with:

- which translation patterns look strongest
- what barriers must be designed around
- what a minimum viable field playbook should contain
