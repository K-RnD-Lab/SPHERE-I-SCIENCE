This file marks the transition from:

- architecture docs

to:

- actual research preparation

### 3. Actual research starter pack

## What comes next

For a research line to move from planning into execution, the next steps are:

1. fill the evidence schema with real records
2. expand the source registry
3. create first figures
4. write the first report draft
5. mirror the stable report into the real `S` folder

## Working rule from now on

Do not keep adding only abstract guides forever.

For each sphere, after enough architecture exists, switch into:

- evidence
- source tracking
- report skeleton
- then real analysis

Planning files that no longer support an active study should move to `archived_docs/`.
