# S4 Biochemistry & Metabolomics

`S4` is the cross-organism chemistry and mechanism lane inside `SPHERE-I-SCIENCE`.

## What belongs here

- cross-organism biochemical mechanisms
- metabolomic signatures and pathway-level inference
- enzyme, substrate, and pathway perturbation logic
- chemistry-facing questions that are not limited to one biological domain

## Example directions

- metabolite signature classifier
- pathway perturbation maps
- enzyme/substrate inference

## Boundary rule

Use `S4` when the main object is a biochemical mechanism or metabolomic pattern. If the domain is clearly plant-intrinsic, agricultural, neuroscience, or ecology-led, place it in `S2`, `S3`, `S5`, or `S6` instead.
